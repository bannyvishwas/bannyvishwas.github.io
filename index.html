Chapter 2
Virtual Assistant(DL)

Virtual assistant, as it's name says, it assist us to do some regular daily tasks. There are lot of different techniques integrated together to make a virtual assistant that can response to your command by voice or text. There are different techniques mentions:
1. NLU(Natural Language Understanding) : Technique to understand, what the user is saying?
2. Intention Detection System : Technique to detect the intent of the command i.e. If user is asking for the time or about weather
3. NLG(Natural Language Generation) : Technique to generate response, it could be in the form of text or voice or both. If voice is to be responed then Text To Speech(TTS) Engine needed to be integrated
